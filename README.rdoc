== 北附人 Oauth2.0 认证系统

此为北附人平台的 Oauth2.0 认证系统源码。

项目地址： http://www.bdfzer.com

==== 软件环境

* Ruby 2.1.1

* Rails 4.1.0.rc1

* Redis

* Postgresql 9.1+

* Semantic-UI

* Devise

* RubyCas-Client (github: 'cassiuschen/rubycas-client', branch: 'feature/UpdateToRails4.1')

* DoorKeeper

==== 功能

* 使用北大附中CAS注册用户

* 使用北大附中学号登陆系统

* 开放的 Oauth2.0 认证

* Oauth2.0 认证下的用户信息API
